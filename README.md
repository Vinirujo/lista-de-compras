# Lista de Compras

Aplicação de terminal em Python para gerenciar uma lista de compras, com menu interativo em loop.

## Funcionalidades

- Adicionar item à lista
- Remover item da lista (com validação se o item existe)
- Exibir a lista de compras numerada
- Sair do programa

## Tecnologias

- Python 3

## Como executar

```bash
python lista_de_compras.py
```

## O que aprendi

Praticar manipulação de listas (`append`, `remove`, `enumerate`), estruturação de menus com `while` e condicionais, e validação de entradas do usuário.
